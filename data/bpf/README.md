## How agents use this

1. Research Analyst reads the latest snapshot → thesis memo
2. Quant reads historical snapshots → backtest evidence
3. CIO reads thesis memo + current snapshot → strategy

Add new snapshots here when a new BPF report is published. The Research Analyst is triggered to produce a new thesis memo on each new snapshot.
