# Profile 1 — Pure Option Book

**Risk level:** Maximum aggressive  
**Option analogy:** Deep out-of-the-money call spread — most expirations are zero; one position covers all losses  
**Target client:** Sophisticated investor with high risk tolerance, treats this allocation as a venture portfolio within a larger wealth structure

## Allocation

| Asset class | Target % | Notes |
|-------------|---------|-------|
| High-conviction thesis plays (HYPE-type) | 60% | 2–4 positions max, each thesis-backed |
| Emerging L1/protocol OTM plays | 40% | 3–6 positions, BPF-qualified only |
| Stablecoin floor | 0% | None required |
| BTC | 0% | Not held — too low gamma |
| ETH | 0% | Not held — too slow |

## Risk parameters

- Max single position: 40%
- Drawdown trigger: -20% → alert, -30% → mandatory CIO review, -40% → principal escalation
- Rebalancing: Monthly, or on thesis invalidation
- Stop-loss: Thesis-based, not price-based (see THESIS.md Axiom 5)

## Expected behavior

In a bull market, this profile should return 3–10x. In a bear market, this profile can lose 80–100% of value. This is understood and accepted by the client. The strategy is: many options, most expire zero, one covers everything.

## Suitability

Client must confirm in writing:
- This allocation represents <20% of total liquid net worth
- Client has read THESIS.md and understands the option framework
- Client does not require this capital within 3 years
